- Create a virtual environment:
python -m venv venv
source venv/bin/activate        # On Linux/macOS
.\venv\Scripts\activate           # On Windows


- Install dependencies:
pip install -r requirements.txt

- Database Config
open __init__.py file and change the Database config 
based on your system (like root name and password)

- Running:
python run.py
